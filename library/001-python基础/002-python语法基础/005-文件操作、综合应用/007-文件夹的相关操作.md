## 文件夹的相关操作
实际开发中，有时需要用程序的方式对文件夹进行一定的操作，比如创建、删除等

就像对文件操作需要os模块一样，如果要操作文件夹，同样需要os模块

---
### <1>创建文件夹

    import os

    os.mkdir("张三")
### <2>获取当前目录

    import os

    os.getcwd()
### <3>改变默认目录

    import os

    os.chdir("../")
### <4>获取目录列表

    import os

    os.listdir("./")
### <5>删除文件夹

    import os

    os.rmdir("张三")
